# Claude Instructions for this Repo

Goal: Keep changes small, safe, and testable.

Rules:
- Ask for a plan before editing files.
- Make minimal changes; avoid big rewrites.
- Don’t add dependencies unless necessary.
- Keep behavior the same unless explicitly requested.
- Add/adjust tests when logic changes.
- Update README/docs when usage changes.

Definition of Done:
- Script runs locally
- Tests pass
- `git diff` reviewed
- Changes committed with clear messages
